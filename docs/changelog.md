## [0.0.1] - 2026-03-28

### Added

- Fetch Merriam-Webster's Word of the Day from RSS
- Structured notes with pronunciation, definition, examples, and etymology
- Osmosis flashcard integration (cloze, basic, bidirectional)
- Deck assignment via frontmatter
- Native Obsidian callouts or Admonitions plugin syntax
- Folder auto-suggest for destination folder setting
- Insert WOTD link at cursor command
- Ribbon icon using `message-square-quote`
