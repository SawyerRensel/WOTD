# Osmosis Integration

Word of the Day integrates with [Osmosis](https://github.com/SawyerRensel/Osmosis), a spaced-repetition flashcard plugin for Obsidian. When enabled, each WOTD note includes a flashcard so you can study and retain new vocabulary.

- [Flashcard Types](flashcard-types.md) -- cloze, basic, and bidirectional cards
- [Decks](decks.md) -- organize your word flashcards into Osmosis decks

!!! info
    Osmosis must be installed and enabled for flashcards to work. See the [Osmosis documentation](https://sawyerrensel.github.io/Osmosis/) for setup instructions.
